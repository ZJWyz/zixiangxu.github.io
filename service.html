Notes on visitor analytics

What is already wired in: the homepage includes a ready-to-enable GoatCounter block. Replace YOURCODE in index.html with your GoatCounter site code to turn on a live visit counter.

Country / region data: GoatCounter stores location codes at country / region level in its analytics data, so you can see where visits come from in the dashboard / export.

Public visitor map: if you want a public map on the homepage, create a widget with a map service and paste its embed code into the homepage. I kept this optional so the site stays clean and privacy-conscious by default.

Prepared as a GitHub Pages-ready multi-page site. The design intentionally uses a restrained, professor-style layout for long-term academic maintenance.
