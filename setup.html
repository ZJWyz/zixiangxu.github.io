How to deploy

  1. Create a public repository named YOUR-USERNAME.github.io.
  2. Upload all files in this folder to the repository root.
  3. Go to Settings → Pages.
  4. Select Deploy from a branch.
  5. Select branch main and folder /(root).
  6. Wait for GitHub Pages to publish the site.

How to enable the visit counter

  1. Create a GoatCounter site.
  2. Open index.html.
  3. Find https://YOURCODE.goatcounter.com/count.
  4. Replace YOURCODE with your real GoatCounter site code.
  5. Commit and push the updated file.
After activation, the homepage will show a visible total-site visit counter. GoatCounter can also record location codes at country / region level in its analytics data.

Optional public visitor map

If you want a visible public map of visitor locations, you can add a separate map widget after the site is live. I left this optional so the homepage stays clean and privacy-conscious by default.

How to edit later

  • For small changes, use GitHub’s in-browser editor.
  • For larger changes, edit the HTML files locally and push again.
  • The site is plain HTML/CSS, so no build system is required.
