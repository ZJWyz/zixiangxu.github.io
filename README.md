# Minimal academic homepage (3 pages)

Files:
- index.html
- publications.html
- activities.html
- style.css

## Deploy on GitHub Pages
1) Create a public repo named `<username>.github.io`
2) Upload these files to the repo root
3) Settings → Pages → Deploy from a branch → main / (root)

## Optional visitor analytics (with country/region)
GoatCounter is lightweight and provides geography in its dashboard.
Uncomment the GoatCounter script in the footer of each page and replace YOURCODE.
